# Barriers to Assistive Technology Uptake Among Persons with Disabilities in Selected Urban Districts in Ghana

This dataset contains anonymised quantitative data collected as part of a cross-sectional study examining the prevalence and barriers to assistive technology (AT) uptake among adults with mobility and visual impairments in four districts in the Ashanti Region of Ghana. The dataset includes responses from 189 participants recruited from disability associations.

The dataset comprises variables on:

- Socio-demographic characteristics (age, sex, education level, employment status).

- Disability type  (mobility impairment or visual impairment).

- Assistive technology use (type of devices).

- Barriers to AT adoption (cost, stigma, etc)

- Overcoming barriers/facilitators (training, advocacy, financial assistance, availability and accessibility etc).

**NB:** All data have been de-identified to ensure participant confidentiality, and no personal identifiers are included
